# #TeamGraduate AI: Supporting Materials

**Project Name**: "Empowering #TeamGraduate Educators: Generative AI Tools for Secondary E-Learning in Namibia"

**Project Description**: #TeamGraduate AI is a WPI IQP project focused on empowering educators in Namibia with generative AI tools for secondary e-learning. This repository contains supporting materials for the project.

The actual code is not publicly available, since we developed this product specifically for #TeamGraduate. We can provide additional details upon request.
